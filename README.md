# Modern E-commerce Website

A beautiful and modern e-commerce website with animations, cart functionality, and a backend server.

## Features Added

1. Modern animations using Animate.css and AOS (Animate On Scroll)
2. Cart functionality with backend integration
3. Responsive product cards with hover effects
4. Smooth scrolling and improved navigation
5. Backend API for product management

## Setup Instructions

1. Install Node.js from https://nodejs.org/
2. Open terminal in the project directory
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   node server.js
   ```
5. Open index.html in your browser

## Technologies Used

- Frontend: HTML5, CSS3, JavaScript (jQuery)
- Backend: Node.js, Express.js
- Animations: Animate.css, AOS
- Icons: Boxicons
